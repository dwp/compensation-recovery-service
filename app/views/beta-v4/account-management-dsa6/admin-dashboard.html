Requesting certificates

You can only request a certificate using this service for claims registered online.

If you use a third party software to register claims, you can use the CRU e-service to request a certificate.

For claims registered by post, send an email to cru.highvolumeareacertificates@dwp.gov.uk to request a certificate. You must include the CRU reference
(3 letters followed by 3 numbers) in your email.
