Summary of design changes

Latest changes

February 2023

  • 07/02/23: NHS Hospitals and Trusts - content review completed and screens updated.
  • 01/02/23: NHS Hospitals and Trusts - Draft version of the CRU Ops journey for managing NHS details (content review pending).

January 2023

  • 16/01/23: Certificate expiry - Compensator journey for requesting a new certificate.
  • 16/01/23: Tasks & workflows - Updates to DA task and new expiring certificate task.

December 2022

  • 16/12/22: Request access - if a company name match is flagged, the user can request access against that company match. Request access now includes passcode screen.
  • 16/12/22: Delegated authority - take over/hand back journeys, claim views for various scenarios, CRU Ops view of task for taken over requests
  • 13/12/22: Ops UI - Tasks and workflows - No NINO task unhappy path and Legacy tasks list

November 2022

  • 18/11/22: Account registration - pagination added to Manage org/Invite user table, changes to request access and token expiry screens (version 2.50)
  • 16/11/22: Further updates to account registration - request access journey for admin user, token expiry screens (version 2.40)
  • 11/11/22: Changes to Account registration to include moving the Company name/match question to the front of the journey and reordering the rest of the flow. 'Request access' flow added. (version 2.30)
  • 08/11/22: Remove user journey added. Change/edit company details added. (version 2.20)
  • 03/11/22: Post-research iterations to account registration journey (version 2.10)

October 2022

  • 31/10/22: New account registration journey (version 2.00)
  • New account registration journey (version 1.01 - content pending)
  • New navigation and views for the compensator dashboard, all claims view, register new claim etc.
  • Journey added for the delegated authority - taking over an existing claim.

August 2022

  • Start page & new claim: Choice of liability added between Public and Motor. Screen and content added to new claim journey
  • New claim: Date of injury page moved forward to be part of the injured persons details

July 2022

  • New claim: Error validation added to simulate preventing 'No' to both whiplash and other injury choices
  • New claim: Manual address capture journey added
  • New claim: Injury capture, location of injury added
  • New claim: Check answers page interactions fixed when changing an answer
  • New claim: Check answers page layout updated for the injury capture data
  • Compensator claim view: Updated for the injury capture data
  • Ops claim view: Updated for the injury capture data
